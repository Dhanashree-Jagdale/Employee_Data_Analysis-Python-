# Employee_Data_Analysis-Python-
Employee data analysis using Python (Pandas, EDA, data cleaning &amp; insights)

## Project Overview
This project focuses on cleaning and analyzing employee data to extract meaningful insights.

## Key Analysis Performed
- Highest paying department
- Highest paying city
- Experience vs Salary
- Hiring trends by year
- Employee distribution by department

## Tools Used
- Python
- Pandas
